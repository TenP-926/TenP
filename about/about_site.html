このサイトについて

長すぎる概要

このサイトは、自分がプログラミングを勉強するために作りました。Githubを使ってみたかったし、Github Pagesとかいう機能があるからどうせなら公開したいと思って公開しました。マイペースで更新していくつもりです。

あとは、工業高校とか高専生になる周りがちょっと羨ましくて、自分もプログラミングしたいなって思って作った。あと春休みの暇つぶし((

もしかしたらドメイン買って脱無料サーバーするかも？ サーバーサイドもやってみたいので。

まずは物置のところの作品をJavaScriptからめて作っていきたい

完成品みて思ったけど、なんか市区町村のホームページにありそうな堅苦しいサイトになってません?

すごく話が点々として申し訳ないですが、どんな環境からどのくらいアクセスがあるのか見てみたいのでGoogle Analyticsを導入するかもしれません。まずそんなアクセス量ないと思うけど。

対応(推奨)環境

このサイトが対応している端末(環境)です。

◇PC

Windows 10,11 (サポート中のバージョン)

  • Mozilla Firefox 最新バージョン
  • Google Chrome, Microsoft Edge 最新バージョン
  • その他、Chromiumベースのブラウザ(Brave, Operaなど)の最新バージョン

MacOS

  • ブラウザが最新バージョンであり、MacOS(サポート中の端末)も最新である環境

Linux(いないと思うけど)

  • サポート中で最新バージョンのもの

ChromeOS

  • サポート中の端末で最新バージョンのもの

◇スマートフォン/タブレット

横幅360px以上の端末をサポートします。それ以下はレアケースってことで許してください...

Chromeの開発者ツールに320pxって文字が見えて、調べたらAndroid端末にそのサイズあったけど、崩れすぎててめんどくさくなったから

Android

  • Mozilla Firefox 最新バージョン
  • Google Chrome, Microsoft Edge 最新バージョン
  • その他、Chromiumベースのブラウザ(Brave, Operaなど)の最新バージョン

iOS

  • ブラウザが最新バージョンであり、iOS(サポート中の端末)も最新である環境
